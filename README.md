# Math University

This repository hosts the frontend and curriculum data for Math University, an accreditable-style curriculum system.

## Getting Started

To view the blog, enable GitHub Pages for this repository from your GitHub settings.

## Data Structure

Curriculum data will be stored in the `data/` directory as JSON files:
- `taxonomy.json`: Canonical Math Taxonomy (Levels 1-3)
- `tracks.json`: Degree Tracks (BS Math, BS Applied Math, etc.)
- `courses.json`: Detailed Course Library
- `prereqs.json`: Course Prerequisites (forming a DAG)
- `assessments.json`: Course Assessments and Mastery Rubrics
